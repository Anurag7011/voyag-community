# **App Name**: WanderHub

## Core Features:

- Location Submission: Users can submit unique locations with captions, images, country, city/town, hashtags, and Google Maps link.
- Location Search: Users can search for locations based on country, city, or town.
- Event Promotion: Locals can upload events (major or low-key) with date, time and place, for travellers to find authentic local experiences.
- AI-Powered Trend Spotting: An AI tool analyzes location data to identify emerging travel trends and off-the-beaten-path destinations.
- User Profiles: Allow travellers to create their profile and upload photos of places they have visited

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2), inspired by clear skies and open horizons, for trust and adventure.
- Background color: Light gray (#F0F4F7), a desaturated tint of the primary color, for a clean and modern feel.
- Accent color: Warm orange (#F2994A), analogous to blue, suggesting energy and discovery for interactive elements.
- Body and headline font: 'PT Sans' for a modern, warm, and accessible look.
- Use clean and outlined icons for navigation and location details.
- Modern, card-based layout inspired by 'Get Your Guide' and 'Viator'.
- Subtle animations when new locations or events are loaded or when the user interacts with maps and location cards.
